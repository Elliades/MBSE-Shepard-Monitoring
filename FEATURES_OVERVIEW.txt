╚══════════════════════════════════════════════════════════════════════════════╝

  1️⃣  Open http://localhost:3000 in your browser

  2️⃣  Look for NEW elements:
      • Statistics Panel (left side, 3rd panel)
      • Action Buttons (left side, 4th panel)
      • ⌨️ Keyboard button (bottom-right corner)

  3️⃣  Try the features:
      • Press ? to see keyboard shortcuts
      • Run a mission: press 1, 2, 3, 4, 5, 6
      • Watch statistics update in real-time
      • Click "Export Logs" to save data

  4️⃣  Test keyboard shortcuts:
      • Use number keys for quick control
      • Press 'r' to reset mission
      • Press '?' to toggle help modal

  5️⃣  Export your data:
      • Export Logs → Human-readable text
      • Export JSON → Structured data
